# 🤖 텔레그램 봇 사용법

봇은 **단방향 알림 + 매물 등록** 용. 봇한테 명령 보내서 처리하는 건 등록 흐름뿐 (수락/거절 등 다른 액션은 웹).

## 0. 처음 연결 (한 번만)

각 admin 마다 한 번씩:
1. 텔레그램에서 본인 봇 검색 → `/start` 입력
2. 봇이 알려주는 `chat_id` 복사
3. 웹사이트 `/settings` → "텔레그램 chat_id" 칸에 붙여넣고 저장 → 🧪 테스트
4. 텔레그램에 "✅ meetcute 테스트 메시지" 도착하면 OK

> 봇 username 모르면 책임자에게 물어보세요. `/settings` 페이지 상단에 🟢 "봇 연결됨: @xxx_bot" 박혀있음.

## 1. 봇 명령어

| 명령 | 동작 |
|---|---|
| `/start` | chat_id 확인 + 전체 가이드 + 양식 자동 전송 |
| `/help` | 명령어 목록 + 알림 종류 + 웹 주요 URL |
| `/me` | 본인 계정 정보 (닉네임 / 권한) |
| `/form` (= `/register`) | 매물 등록 양식 받기 |
| `/done` | 사진 단계에서 등록 마무리 |
| `/cancel` | 진행 중인 등록 취소 |

## 2. 매물 등록 양식

### 흐름
```
1. /form           — 봇이 빈 양식 보냄
2. (양식 복사 → 빈칸 채워서) → 한 번에 답장
3. 봇이 검증 → ✅ 입력 확인
4. 사진 1~5장 전송
5. /done           — 자동 등록 완료
```

### 양식 (자유 포맷)
```
성별: F
이름: 민지
사는곳: 서울 마포구
직장: ○○회사 마케팅팀
나이: 28
키: 165
(선택)
이상형: 차분한 사람
메모: 친구 추천
```

- 순서 자유 / 추가 줄 자유 / 한글 라벨 자유 (`거주지`, `회사`, `별칭`, `신장` 등도 인식)
- 빈 칸 / 잘못된 값 있으면 → ❌ 어떤 줄이 문제인지 알려줌 → 수정해서 다시 보내면 됨
- 이상형 / 주선자 메모는 양식에서 빠뜨려도 OK (등록 후 웹 `/persons/{id}/edit` 에서 추가)

### 사진
- jpg/png/webp/heic 지원
- 모바일 사진의 EXIF 회전 자동 적용
- 최대 5장 (초과는 무시)
- 사진 없이 `/done` 도 가능

## 3. 봇이 보내는 알림 종류

| 이벤트 | 받는 사람 | 메시지 |
|---|---|---|
| 새 소개 요청 도착 | 받는 admin | 📨 보낸 분 / 내 매물 / 소개 매물 / 메모 |
| 보낸 요청 수락됨 | 보낸 admin | ✅ 응답한 분 / 자동 생성된 만남 기록 링크 |
| 보낸 요청 거절됨 | 보낸 admin | ❌ 사유 |
| 받은 요청 취소됨 | 받는 admin | ↩️ 취소한 분 |
| 24h+ 미응답 PENDING 요청 | 받는 admin | ⏰ 매일 1회 재알림 |
| **공개 URL 변경** | 등록된 모든 마담뚜 | 🔄 새 URL + "홈 화면 다시 추가" 안내 |

> 본인이 한 행동(매물 등록 / 만남 추가)은 알림 안 보냄 (노이즈 방지).

## 4. 자주 묻는 것

### 채팅 나가면 chat_id 사라져?
**아니요.** 텔레그램 영구 ID. 채팅 닫아도 / 메시지 지워도 / 봇 차단해도 chat_id 그대로.
- 차단했으면 봇 메시지가 안 옴 → 차단 해제 + /start 한 번
- 다른 폰으로 로그인해도 같은 계정이면 chat_id 동일

### 양식 검증 실패하면?
검증 실패한 항목 목록이 와요. 양식 다시 받지 말고 그냥 같은 메시지 수정해서 재전송하면 됩니다 (`/cancel` 안 필요).

### 사진 거꾸로 안 떠?
EXIF orientation 자동 처리됨. 거꾸로 뜨면 알려주세요 (특수한 메타데이터일 수 있음).

### 봇이 메시지에 답장 안 해
봇은 **수동 명령만** 처리. /start, /form, /done, /cancel, /me, /help 외엔 무시. 매물 수정/조회 같은 건 웹에서.

### 그룹 채팅에서 쓰고 싶어
**1:1 채팅만 지원.** 그룹/채널에선 작동 안 함 (개인정보 보호).

## 5. 봇이 못 하는 것 (웹에서만)

- 매물 수정/삭제
- 만남 기록 추가/수정/결과 변경
- 소개 요청 보내기 (다른 마담뚜의 매물한테)
- 받은 요청 수락/거절
- 호환성 체크
- 유저 권한 관리
- 통계 / 변경 이력 조회
